# Damu — Film Site Admin

**English** | [简体中文](./README.zh-CN.md)

Backend admin for a film/TV portal, built on the RuoYi framework. SQL scripts are included under `sql/`.

| Platform | URL |
|----------|-----|
| **GitHub** | https://github.com/xiaoyutoucom/damu |

## Features

- RuoYi-based admin modules
- Film/TV portal backend APIs & pages
- SQL init scripts under sql/

## Tech stack

- Java · Spring Boot · RuoYi · Maven

## Build & run

Import SQL from `sql/`, configure datasource in the RuoYi admin module, then start with `ry.bat` / `ry.sh` or run the Spring Boot admin module from your IDE.
