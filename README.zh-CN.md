# 大幕 — 影视官网后台

[English](./README.md) | **简体中文**

影视官网后台管理系统，基于若依（RuoYi）框架二次开发。数据库脚本在 `sql/` 目录。

| 平台 | 地址 |
|------|------|
| **GitHub** | https://github.com/xiaoyutoucom/damu |

## 功能

- 基于若依的后台模块
- 影视官网后台能力
- sql/ 目录含数据库脚本

## 技术栈

- Java · Spring Boot · RuoYi · Maven

## 编译运行

导入 `sql/` 中的数据库脚本，配置数据源后，用 `ry.bat` / `ry.sh` 或 IDE 启动 Spring Boot 管理端。
